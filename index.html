Sobre mim

Apaixonado por tecnologia
e código limpo

Atualmente cursando Análise e Desenvolvimento de Sistemas, dedico-me a construir uma base sólida em engenharia de software e na resolução criativa de problemas. Minha trajetória sempre foi guiada por uma curiosidade genuína pelo universo da tecnologia.

O primeiro contato com HTML e CSS abriu meus olhos para um mundo vasto e fascinante. Contudo, foi ao descobrir o JavaScript e escrever o meu primeiro "Hello World" que senti o desejo definitivo de transformar essa paixão em profissão, o que me motivou a ingressar na faculdade.

Desde o início da minha formação acadêmica, venho trilhando uma jornada de aprendizado contínuo. Comecei aprimorando minha lógica de programação com Portugol e C, evoluindo para o desenvolvimento de aplicações práticas com Java e JavaScript — incluindo a criação de pequenos jogos web.
